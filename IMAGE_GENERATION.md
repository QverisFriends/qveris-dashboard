# AI 图片生成功能使用指南

## ✅ 功能状态

**状态**: 正常工作
**API**: 智谱AI CogView-3
**响应时间**: 5-10秒

## 🚀 如何使用

### 步骤1: 访问页面
打开浏览器访问: http://localhost:9000

### 步骤2: 找到AI图片生成卡片
在页面下方找到 "AI 图片生成" 卡片

### 步骤3: 输入描述
在文本框中描述你想要生成的图片，例如：
- "一只橙色的小猫在草地上玩耍"
- "日落时分的海滩风景"
- "赛博朋克风格的城市夜景"
- "可爱的小熊猫吃竹子"

### 步骤4: 选择尺寸
- 512x512 (快速生成，适合测试)
- 768x768 (中等质量)
- 1024x1024 (高清质量，耗时较长)

### 步骤5: 点击生成
点击 "生成AI图片" 按钮，等待5-30秒

### 步骤6: 查看结果
- 图片生成后会显示在卡片下方
- 点击图片可以查看大图
- 右键可以保存图片

## 🔧 技术细节

- **模型**: CogView-3
- **最大长度**: 约500字符的描述
- **支持语言**: 中文、英文
- **生成时间**: 5-30秒（取决于尺寸）
- **格式**: PNG/JPEG

## 🛠️ 调试

如果遇到问题：

1. **检查控制台**: 按 F12 查看详细错误
2. **简化描述**: 尝试更短、更简单的描述
3. **选择小尺寸**: 512x512 生成最快
4. **等待**: 有时API响应较慢，请耐心等待

## 📝 示例代码

如果你想通过API直接调用：

```python
import requests

API_KEY = 'your_api_key'
url = 'https://qveris.ai/api/v1/tools/execute?tool_id=bigmodel.images.generation.create.v1'

payload = {
    'search_id': '',
    'session_id': '',
    'parameters': {
        'model': 'cogview-3',
        'prompt': '一只可爱的小猫',
        'size': '512x512'
    },
    'max_response_size': 102400
}

response = requests.post(url, json=payload,
    headers={'Authorization': f'Bearer {API_KEY}', 'Content-Type': 'application/json'})

data = response.json()
image_url = data['result']['data']['data'][0]['url']
print(f"生成的图片: {image_url}")
```
